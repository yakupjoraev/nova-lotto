Особенности адаптивного тултипа:

  • Автоматическое позиционирование в зависимости от доступного места
  • Закрытие по клику вне тултипа
  • Закрытие по кнопке крестика
  • Закрытие по клавише Escape
  • Адаптивные размеры для мобильных устройств
Этот тултип должен открываться слева от кнопки, если есть место. Если места нет, он адаптивно изменит свою позицию.
